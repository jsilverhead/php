Строки можно конкатинировать путём сложения:
```go
func Main() {
firstName := "John"
lastName := "Smith"

fullName := firstName + " " + lastName

fmt.Println(fullName)
}
```

Также строки можно объединять с помощью sprintf:

```go
func Main(){
firstName = "John"
lastName = "Smith"

fullName = sprintf("%s %s", firstName, lastName)

fmt.Println(fullName)
}
```
