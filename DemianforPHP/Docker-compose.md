### Docker

Команды:

- Стартует контейнер:
```
docker start <name/id>
```

- Ставит контейнер на паузу:
```
docker pause <name/id>
```

- Останавливает контейнер:
```
docker stop <name/id>
```

- Выводит информацию о существующих контейнерах:
```
docker info
```

- Выводит помощь:
```
docker help
```

Выводит справку о команде:
```
docker <команда> -help
```
